# The Ultimate Data Science and Basic GenAI Bootcamp

Welcome! This is your companion repository for **The Ultimate Data Science and Basic GenAI Bootcamp**—a hands-on, project-focused journey to build a strong foundation in data science, machine learning, and generative AI.

## 📚 What This Bootcamp Covers

You'll explore and practice:
- **Data Preprocessing**  
  Cleaning, transforming, and preparing data for analysis.
- **Exploratory Data Analysis (EDA)**  
  Discovering patterns and insights using statistics and visualization.
- **Machine Learning Algorithms**  
  Building and evaluating supervised and unsupervised models.
- **Deep Learning Fundamentals**  
  Understanding the basics of neural networks and their operations.
- **Introduction to Generative AI**  
  Getting started with foundational GenAI concepts and workflows.

## 📁 Repository Layout

Here's how the course materials are organized:

```
.

├── 3. Machine Learning Algorithms/
├── 4. Deep Learning Fundamentals/
```

Each numbered folder matches a module in the bootcamp. Explore them in order or revisit them as needed.

## 🧑‍💻 How to Use This Repository

1. **Clone the Repository**

    ```sh
    git clone https://github.com/your-username/ultimate-ds-genai-bootcamp.git
    cd ultimate-ds-genai-bootcamp
    ```

2. **Set Up Your Python Environment**

   - Install dependencies:
     ```sh
     pip install -r requirements.txt
     ```
   - Using a virtual environment (e.g., Anaconda or venv) is recommended.

3. **Work Through the Notebooks**

   - Open Jupyter or VSCode and navigate to each module.
   - Read theory, run code examples, and attempt exercises and challenges.

4. **Use the Datasets**

   - Practice with sample datasets provided in the `datasets/` folder.
   - Feel free to add your own data for experimentation.
